# 용어 (Terminology)
## 상계 and 하계 and 유계
* A: 실수의 부분집합, x: all element in A, a, b: 실수
* x <= a를 만족하는 a ==> A의 상계
* b <= x를 만족하는 b ==> A의 하계

### 유계
* 위로 유계: A가 상계를 가짐
* 아래로 유계: A가 하계를 가짐
* 유계: A가 상계, 하계를 모두 가짐

### 상한, 하한
* a: A의 상계, a' < a 이면 a'는 A의 상계가 아님
* ==> a: A의 상한(= 최소상계)

* b: A의 하계, b < b'이면 b'의 A의 하계가 아님
* ==> b: A의 하한(= 최대상계)
