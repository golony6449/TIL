# 10.6 멱급수
## 용어
* 멱급수: 변수(x)의 거듭제곱들을 포함하는 급수 ex) Sum(An * x^n)
* 초월함수: 다항식으로 구성되지 않는 함수. ex) 지수함수, 삼각함수, 로그함수

* 초원함수는 적당한 지점에서 잘라서 근사치 사용

## 목표
1. 대응하는 멱급수 찾기
2. 수렴성 여부 판단 (위에서 언급된 5가지 방법 사용)

## 사용하는 이유
* 적분의 용이함

## 수렴성 여부 판단
### Case 1
* 조건: Sum(An * x^n) 이 x=c에서 수렴, x=d에서 발산
* 결론
* |x| < |c|를 만족하는 모든 x에서 절대수렴
* |x| > |d|를 만족하는 모든 x에서 발산

### Case 2
* 조건: Sum(An * (x-a)^n) 에서
* 결론
* <b>x=a 일때만</b> 수렴
* 모든 실수 x에서 절대수렴
* |x-a| < R일때 절대수렴, |x-a| > R일때 발산하는 양수 R이 존재 ( |x-a| = R => 수렴 발산 모두 가능 ) 
* (R: 급수의 수렴반경, 수렴구간 길이의 절반) (수렴구간: 급수가 수렴하는 x의 집합)

## 멱급수의 미분적분
* 조건: Sum(An * x^n), 수렴반경 r
* 항별 미분 결과( Sum(n * An * x(n-1) ) )의 수렴반경: r
* 항별 적분 결과( Sum( (1/(n+1) * An * x^(n+1) ) ) )의 수렴반경: r

* 급수의 미분 == 점화식의 미분
* 급수의 적분 == 점화식의 적분

##  테일러급수 and 테일러정의
### 용어
* 테일러 다항식: f(0) + f'(0)x + f''(0) * x^2/2! + ... + f(n)(0) * x^n / n! + f(n+1)(c) * x^(x+1) / (n+1)!
* (판서): Sum( f(n)(a) * (x-a)^n / n! )

* k차 테일러 계수: f(k)(0) / k!
* (판서): f(k)(a) / k!
* 매크로린 급수: a=0인 테일러급수

* 잉여항: f(n+1)(c) * x^(n+1) / (n+1)!

### 테일러 정리
* 조건: 함수 f가 0, x를 포함하는 열린구간에서 정의, f의 n+1차 도함수 존재, 연속
* 결론: 테일러 다항식을 만족하는 c가 [0, x]에 존재

### 사용
* 적분이 쉽지않은 경우 사용
