# 부팅과정

## 과정

1. Power on (or PC Reset)
2. ROM BIOS program test and initialize HW
3. ROM BIOS Boot program Select a boot devices
4. Read theMBR into Memory
5. MBR select active partition
6. Read PBR(or VBR) from partition
7. PBR load bood loader
8. Kernel load shell
