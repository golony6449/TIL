## Vmware에서 네트워크 설정
==========================
* 종류: Bridged, NAT, Host-only
* Player는 vmnetcfg(네트워크관리자)가 없음 -> '필요하면' 인터넷에서 Workstation에서 추출한 파일을 다운로드 받아 사용

### Bridged
* 게스트 OS가 인터넷에 직접 연결됨

#### 설정
1. 가상머신 설정에서 네트워크 탭으로 이동
2. Bridged 선택 (실제 네트워크의 상태 연동 옵션은 필요시 체크)
3. NIC가 여러개(노트북이라던가... 노트북이라던가... 노트북이라던가...)인 경우 auto를 사용해도 되지만, 사용 할 NIC를 지정해 주는 것이 좋아보임

#### 특이점
* vmnetcfg에서 Bridged로 Vmnet을 설정하면 네트워크 어댑터에서 사라짐 -> 고정 IP 필요시, 게스트에서 고정 IP로 설정

### NAT
* Host가 공유기 역할을 수행 -> Host의 IP를 공유

#### 설정 - VM 생성시 기본 설정
1. 가상머신 설정에서 네트워크 탭으로 이동
2. NAT 선택


#### 특이점
* 포트포워드 기능 필요시, vmnetcfg에서 설정

### Host-only
* 호스트와의 통신만 가능 -> 외부망 접속 불가

#### 설정 - VM 생성시 기본 설정
1. 가상머신 설정에서 네트워크 탭으로 이동
2. Host-only 선택
