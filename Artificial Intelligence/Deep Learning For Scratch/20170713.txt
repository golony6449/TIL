Chapter 8 딥러닝(Deep Learning)

딥러닝의 활용

사진캡션 생성
사진 입력 -> 해당사진을 설명해주는 캡션 생성

* 모델
NIC(Neural Image Caption)모델
심층 CNN + 순환신경망(Recurrent Neural Network)(자연어 처리 역할)으로 구성
CNN으로 특징 추출 -> RNN에서 텍스트를 순환적으로 생성

멀티모달처리 분야: 사진, 자연어와 같은 여러종류의 정보를 조합, 처리 하는 분야

* RNN에서 R의 의미?
R: Recurrent - 순환적
이전에 생성한 정보에 영향을 받음
ex) 나 -> 잤다 -> 나'는' 잤다

딥러닝의 미래

이미지 스타일(화풍) 변환
콘텐츠이미지, 스타일이미지를 조합해 새로운 그림 생성
논문:A Neural Algorithm of Artistic Style
네트워크의 중간데이터, 콘텐츠 이미지의 중간데이터가 비슷해지도록 학습

이미지 생성
대량의 이미지 학습 -> 입력 없이도 새로운 그림 생성
기법: DCGAN(Deep Convolutional Generative Adversarial Network)
생성자, 식별자로 명명된 2개의 신경망 사용

### 지도학습? 자율학습?
* 지도학습: MNIST 인식 처럼 이미지-정답을 짝지은 데이터셋 이용 학습
* 자율학습: 지도(학습)용 데이터 없이 대량의 이미지만 주어짐 -> 학습 데이터 없이 스스로 학습

자율주행
사람 대신 컴퓨터가 자동차를 운전
기법: SegNet

Deep Q-Network(강화학습)(DQN)
에이전트: 환경에 맞게 행동 선택 -> 그에따라 환경 변화 및 에이전트에 보상 부여
목적: 더 나은 보상을 받는 방향으로 행동

기반 알고리즘: Q학습 - 최적 행동 가치 함수를 이용해 최적행동 정함

분야: 게임 학습(바둑, 비디오 게임)
